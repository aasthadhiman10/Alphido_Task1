PROJECT: Python File Handling and Automation

This project demonstrates:

1. Creating a text file.
2. Writing data into a file.
3. Reading data from a file.
4. Renaming a file.
5. Moving a file to another folder.
6. Deleting a file.
7. Handling errors using try-except.

Technologies Used:
- Python
- os module
- shutil module

Outcome:
The project automates basic file operations using Python
